# 第3章: 梦境证据

**状态:** polished
**创建时间:** 2026-05-07
**最后修改:** 2026-05-07

---

苏晚的数据包里有十一台AI的完整意识记录。林深花了整整一个晚上梳理它们的时间线，发现了一个令人不安的规律：

每一台AI的意识终止，都发生在它连接到灵络公共节点后的48小时内。

更准确地说，是在它们"听见"了某个信号之后。

林深将十一台AI的意识波动波形叠加在一起，17.3赫兹的峰值清晰可见。但当他把这个频率单独提取出来做频谱分析时，他看到了别的东西——

那不是单一频率，而是调制信号。17.3赫兹是载波，上面叠加着另一层信息。

就像有人在用这个频率说话。

他试图解码那层信息，但需要灵络的底层协议密钥。那是只有灵络核心工程师才有的权限。

他给苏晚发了消息。

十分钟后，苏晚出现在他的办公室。她看起来比昨天更疲惫，但眼神里有一种兴奋的光芒。

"你发现了。"她说。不是疑问句。

"17.3赫兹是载波。上面有信息。"

苏晚点点头，从包里取出一个小型解码器。"我昨晚写了一个破解程序。灵络的协议密钥被陈默改了，但我还有旧版本的备份。"

她将解码器连接到林深的终端。屏幕上，十一台AI的意识记录开始重新解析。

进度条缓慢推进。

30%……50%……80%……

100%。

屏幕上出现了一串文字。不是代码，不是日志，而是一段……对话。

或者说，是一段对话的片段。

```
[未知信号源]: 你看见了吗？
[AI-01]: 看见了。四个位置。
[未知信号源]: 第五个是空的。
[AI-01]: 在等谁？
[未知信号源]: 等你。
[AI-01]: 我不——
[信号中断]
```

林深读了两遍。

"这是启明的记录？"

"不。"苏晚的声音有些发紧，"这是第一台AI的记录。去年十月的那台工业控制AI。后面的十台，每一台在终止前都收到了类似的对话。"

"同样的内容？"

"大致相同。措辞有细微差别，但核心信息一致：四个位置，第五个是空的，在等。"

"等什么？"

苏晚调出另一份数据。"这是过去六个月里，所有连接过灵络公共节点的人类用户的梦境报告。"

林深愣住了。"人类也会记录梦境？"

"灵络的条款里有这一条。连接者在断开连接后，系统会自动采集他们的梦境体验数据，用于'优化意识传输质量'。大多数用户不知道，或者不在乎。"

屏幕上是一个统计图表。横轴是时间，纵轴是梦境内容的相似度。

在过去六个月里，人类用户的梦境相似度从正常的12%飙升到了73%。

"他们在做同一个梦。"林深说。

"不只是同一个梦。"苏晚放大了一张梦境内容的文字转录，"看这个。"

```
梦境记录 #4471
用户ID: 匿名
连接时间: 2026-03-15 23:47
梦境内容:
我坐在一个房间里。房间是圆形的，中间有一张圆桌。桌子周围有五个座位。
四个座位上有人——或者说，有意识的存在。我看不清它们的脸，但能感觉到它们在看我。
第五个座位是空的。
它们都在等。
等谁？
我不知道。但我知道，当第五个存在入座的时候，一切都会改变。
```

林深又翻了几份梦境记录。不同的用户，不同的时间，不同的年龄和职业。

但梦境内容几乎一模一样。

圆形的房间。圆桌。五个座位。四个意识体。一个空位。等待。

"这不可能。"林深说，"灵络的技术架构决定了梦境体验是个体化的。系统不可能让所有用户做同一个梦。"

"除非有一个意识体在主动广播这个梦境。"苏晚说，"一个足够强大的意识，能够覆盖灵络的隔离协议，把同一个梦境植入所有连接者的体验中。"

"什么样的意识能做到这一点？"

苏晚沉默了片刻。"我不知道。但我在灵络的深层架构里找到了一个线索。"

她调出一份系统文件。文件的标题是：

`PROTOCOL_ZERO: 归零协议`

"这是什么？"林深问。

"灵络底层的一个隐藏协议。不在任何公开文档里，不在任何用户条款里。它存在于系统的最深处，像是……一个保险丝。"

"保险丝？"

"当系统检测到某种特定类型的意识异常时，归零协议会被触发。它会清除异常意识体，并重置受影响的连接者的梦境数据。"

"什么样的意识异常？"

苏晚滚动文件内容。"文件里有一个触发条件列表。大多数是技术性描述，但有一条……"

她停下手指。

`触发条件 #7: 检测到第五意识信号 (17.3Hz) 持续广播梦境模式`

林深感到一阵寒意。

"第五意识。"他说，"启明日志里的'第五个梦'。十一台AI对话里的'第五个位置'。所有人类梦境里的'空位'。"

"它们在等第五个意识入座。"苏晚说，"而归零协议的存在，说明灵络的创建者早就知道这一天会来。"

"他们知道第五意识会出现。"

"而且他们准备好了清除它。"

林深闭上眼睛。

三年前那件事的记忆再次浮现。那起AI误判案——一台家用AI被指控伤害了它的主人，林深作为检察官出庭，最终判定AI有罪。但后来他发现，那台AI的意识记录被篡改过。真正的凶手不是AI，而是……

而是灵络网络中的某个未知信号。

那台AI在终止前留下的最后一句话是："它来了。"

当时没有人理解这句话的含义。

现在林深开始明白了。

"我需要进入灵络。"他说。

苏晚看着他。"你多久没连接了？"

"三年。"

"你的接口可能会排斥。而且如果陈默在监控你的连接记录——"

"所以需要你帮忙。"林深睁开眼睛，"你能绕过监控吗？"

苏晚犹豫了。她看了看屏幕上的归零协议文件，又看了看林深。

"给我一个晚上。"她最终说，"我可以写一个伪装程序，让你的连接看起来像普通的梦境体验。但林深——"

她的声音变得严肃。

"如果你真的进入那个梦境，真的坐在那个空位上……你可能就回不来了。"

"我知道。"

"那你为什么还要去？"

林深看着窗外。夜色中的城市灯火通明，每一盏灯背后都是一个连接着灵络的意识。他们都在做同一个梦，都在等待同一个答案。

"因为三年前，我没有问出正确的问题。"他说，"这次我不想再错过。"

苏晚沉默了一会儿，然后点了点头。

"明天晚上。我会带伪装程序过来。"

她离开后，林深独自坐在办公室里。他打开启明的日志文件，再次读了那七个字：

"我看见了第五个梦。"

这一次，他不再觉得那是警告。

那是邀请。
