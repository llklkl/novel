# 审阅模板

## 章节审阅记录格式

审阅过程中，每章记录以下信息（内部笔记，不输出）：

**章节编号**: XX
**身份**: [身份名称]
**关注点记录**:
- [关注点1]: [观察记录]
- [关注点2]: [观察记录]
- [关注点3]: [观察记录]

**章节印象**: [一句话总结该章感受]

```json
{
  "persona_id": "casual-reader",
  "persona_name": "休闲读者",
  "scores": {

  },

  "rating": "推荐"
}
```
